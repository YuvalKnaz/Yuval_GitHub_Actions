# Programming Languages Website

A modern, beautiful static website exploring the differences between programming languages.

## Pages

- **Home** (`index.html`): Introduction to programming languages and their importance.
- **Language Differences** (`differences.html`): Comparisons of popular languages like Python, JavaScript, Java, and C++.
- **Resources** (`resources.html`): Curated learning resources organized by language.

## Features

- Modern design with gradients, shadows, and smooth transitions
- Responsive layout using CSS Grid and Flexbox
- Google Fonts integration for better typography
- Card-based layout for language comparisons
- Hover effects and animations

## Navigation

Each page has navigation buttons to switch between sections seamlessly.

## How to View

Open `index.html` in your web browser to explore the site.

## Technologies Used

- HTML5
- CSS3 (with modern features like CSS Variables, Grid, Flexbox)
- JavaScript (minimal, for navigation)
